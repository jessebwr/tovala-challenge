Centroid calculating database (In US/Canada)

Calculate centroid

To calculate the centroid of all addresses use this widget, or construct a curl request of the form:
curl http://WHATEVERHOSTHISIS/db/api/v1.0/tasks/centroid

Inserting a csv file

To insert a csv file use this widget, or construct a curl request of the form:
curl -X POST -F "data_file=@whatever.csv" http://WHATEVERHOSTHISIS/db/api/v1.0/tasks/insert_csv

Please note that if the entry (by name) is already present it will NOT overwrite it, this will be shown in the return response. ALSO, this csv file must be in the format specified:
name address_line1 address_line2 country state city zipcode

Inserting a single address

To insert a single address use this widget, or construct a curl request of the form:
curl http://WHATEVERHOSTHISIS/db/api/v1.0/tasks/insert/NAME/ADDRESS


NAME is a string, Address is a string of the form: "4100 Dundee Dr., Los Angeles, CA, USA 90027"
but remember to replace all the spaces with %20 (only for the curl request, not this widget..)
